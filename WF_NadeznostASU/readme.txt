Инструкция пользователя


Задача 1

1.	Запустите исполняемый файл WF_NadeznostASU.exe
2.	Перейдите на вкладку "Задача 1"
3.	Выберите необходимые устройства
4.	Введите количество элементов для каждого выбранного вами элемента
5.	Введите время в часах

После этого значения вероятности бесперебойной работы в течение введенного времени и среднего 
времени бесперебойной работы рассчитаются автоматически.


Задача 2

1.	Запустите исполняемый файл WF_NadeznostASU.exe
2.	Перейдите на вкладку "Задача 2"
3.	Введите исходные данные

После этого значения вероятности безотказной работы за время t и delta t, частоты 
и интенсивности отказов на интервале delta t рассчитаются автоматически.


Задача 3

1.	Запустите исполняемый файл WF_NadeznostASU.exe
2.	Перейдите на вкладку "Задача 3"
3.	Нажмите на кнопку "Сгенерировать значения" и введите цифры своего шифра чтобы 
	сгенерировать значения интенсивностей отказов элементов и их количество на каждом 
	слое автоматически или введите их вручную в соответствующих полях

После этого значения средней наработки до первого отказа устройства Tср и вероятности его 
безотказной работы в течение 100 часов Pc(100) рассчитаются автоматически.


В случае проблем с отображением элементов интерфейса программы совершите следующие действия:

1.	Закройте программу
2.	Щелкните правой кнопкой мыши по исполняемому файлу WF_NadeznostASU.exe
3.	В появившемся меню выберите кнопку "Свойства"
4.	В появившемся окне перейдите на вкладку "Совместимость"
5.	Нажмите на кнопку "Измениить параметры высокого DPI"
6.	Поставьте галочку напротив пункта "Переопределите режим масштабирования высокого разрешения"
7.	Убедитесь, что в выпадающем списке под надписью "Масштабирование выполняется" выбран пункт 
	"Приложение"
8.	Запустите программу заново

После этого программа должна отображаться корректно.
